PORTAL DIGITAL SK OBAH — MULA DI SINI

1. Buka PANDUAN-DEPLOYMENT.md.
2. Masukkan konfigurasi Firebase dalam assets/firebase-config.js.
3. Terbitkan firestore.rules dan storage.rules dalam Firebase Console.
4. Salin semua kandungan folder ini ke root repositori portal-sk-obah.
5. Commit dan Push melalui GitHub Desktop.
6. Uji index.html (awam) dan admin.html (panel).

Jangan padam folder assets atau ubah susunan fail.
